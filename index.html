HTML

  • HTML stands for hypertext markup language
  • We can enter many types of HTML to include text, images, and even links in our websites/ apps
  • The head element contains information about the webpage.
  • We connect CSS pages to the head element by adding a link
  • We connect Java pages to the end body element by adding a script tag
  • The body element represents the visible content shown to the user.
  • Text can be broken with a "br" element
  • Text size (usually in the header) can be modified to large by "h1"
  • Text size (usually in the body) can be modified to small by "h6"
  • HTML pages require head, body, main, and footer

CSS

  • CSS stands for cascading style sheet
  • Here, we can modify what our webpage/ app looks like
  • We can modify text styles, color, image borders, liks, etc.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Margin, border, padding, content are within CSS box
  • We can add inline and internal CSS within the html file
  • Most common form of CSS is external, creating a diff. file to edit visuals without direct or permanent affect

Git

  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git checkout plus branch name: will send you to the branch you would like to work in
  • Git version control: add, commit, and push up code in feature branch
  • To do so, first send a pull req. with feat. branch as comparison branch and the default branch as the base branch
  • Then merge PR so main branch is updated
  • In the terminal, pull remote branch into local main branch by entering 'git pull origin main'
  • Add new file tabs like CSS and Java by entering "'touch' and 'name of file''' you want to create (ex: touch style.css)

JavaScript

  • Adds interactivity to our website (clicking buttons, retrieving data, fill out forms, etc.)
  • A javascript variable is a container of data (think of box lbeled "shape" and a sphere in it)
